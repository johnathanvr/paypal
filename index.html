How to deploy sneaky-imagen on RunPod
  1. Log in at runpod.io → Serverless → New Endpoint
  2. Click Browse Hub, search for sneaky-imagen by Sneaky2x, select it
  3. Choose a GPU — RTX 3090 / A4000 for 512 px, A100 for 1024 px+
  4. Set Min Workers: 0, Max Workers: 1 → Deploy
  5. Copy your Endpoint ID from the dashboard (format: abc123xyz456)
  6. Generate an API key at runpod.io/console/user/settings → API Keys
  7. Paste both into the Settings panel above and click Save
Cold start: first request after inactivity takes 30–90 s while RunPod spins up a worker.

Artistic Mode tip: if content is still flagged after enabling Artistic Mode, the worker has a hard-coded filter that can’t be overridden via params. Contact the endpoint owner (Sneaky2x on RunPod Hub) to request a build with safety_checker=false fully supported.
